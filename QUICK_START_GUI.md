# Quick Start: AGI GUI

## Run the GUI

```bash
python agi_gui.py
```

That's it! The GUI window will open.

## Basic Usage

1. **Type a message** in the input box
2. **Press Ctrl+Enter** (or click "Send") to send
3. **View the response** in the chat area
4. **Watch the physiological state** update in real-time

## Teaching Emotions

1. Send a message
2. **Drag the feedback slider** (left = negative, right = positive)
3. Click **"Apply Feedback"**
4. The agent learns that context â†’ emotion association

## Save Your Work

Click **"Save State"** to save the agent's memories and learned emotions to files.

## Features Overview

- **Chat Interface**: Interactive conversation with the AGI
- **Physiological State**: See valence, arousal, heart rate, stress, energy
- **Memory Stats**: Track STM/LTM items and maturity
- **Emotion Feedback**: Teach the agent through feedback
- **State Persistence**: Save/load agent state

See `GUI_README.md` for detailed documentation.
